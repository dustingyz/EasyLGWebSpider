此处的文件，主要用于程序的读取，也可以修改内容改变部分参数。
但是请一定要注意！
所有的文件都是使用 UTF-8 无 BOM 格式进行编码，由于目前没有保存中文，所以可能会被识别 ACSii。
因此不要用 Windows 自带的 notepad 编辑然后保存：
    1. 如果添加了中文，可能会被修改为 GBK 编码
    2. notepad 的 UTF-8 默认添加了 BOM 头，一定会造成程序的错误，因为我懒得加 BOM 处理。


文件描述：
    /dict           自定义词典，可以自行添加，按照 jieba 库的词典格式。
    /http_header    http 请求头。
    /name_map       去重使用的，左边的 Key 一定要是大写。
    /font.TTF       输出词云时显示的字符字体，没有会自动寻找 Windows 自带字体，由于我的是日文系统，所以默认的字体可能会找不到而报错。
    /mask.png       生成词云时使用的蒙版，请自行替换。
